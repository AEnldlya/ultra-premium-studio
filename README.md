# Ultra Premium Studio

A premium multi-page website with cinematic animations and 3D geometric shapes.

## Features

- 5 Pages: Home, About, Services, Portfolio, Contact
- 3D Animated Geometric Shapes (React Three Fiber)
- GSAP Scroll Animations
- Framer Motion Transitions
- Premium Dark Theme with Gold Accents

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- GSAP
- Framer Motion
- React Three Fiber

## Deployment

Connected to Vercel for automatic deployments.
